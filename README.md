## Business Problems Solved
1. Revenue Leakage
2. Cancellation Patterns  
3. Operations Waste

## Tools Used
Excel | SQL | Python | Power BI

## Key Findings
- Room Type A = 45% of total revenue
- Online TA + Long lead time = highest cancellation risk
- August = highest revenue month
